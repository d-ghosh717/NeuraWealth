# 📈 NeuraWealth – AI Powered Stock Market Analysis Platform

NeuraWealth is an AI-driven stock market platform that provides **market insights, predictions, sentiment analysis, and an intelligent chatbot** to help users understand and analyze stock trends in a simple and interactive way.

---

## 🚀 Overview

NeuraWealth is designed as a **smart financial analysis tool** that combines:

* Real-time market understanding
* AI-based predictions
* Strategy testing (backtesting)
* News and sentiment insights
* Chat-based interaction

The goal is to convert complex market data into **clear, actionable insights** for users.

---

## 🧠 Core Features

### 📊 Market Overview

* Displays key market data such as:

  * Indices (NIFTY, SENSEX)
  * Top Gainers & Losers
  * Market trends
* Helps users quickly understand overall market conditions

---

### 🔮 AI Prediction

* Uses historical data and indicators to:

  * Predict short-term stock movement
  * Show expected price trends
* Provides **Buy / Hold / Sell suggestions**
* Includes reasoning based on data

---

### 📉 Backtesting

* Allows testing of simple trading strategies
* Uses past data to check:

  * Profit / Loss
  * Strategy performance
* Helps users understand what works before real investing

---

### 📰 News & Sentiment Analysis

* Displays financial news related to stocks
* Analyzes sentiment:

  * Positive 🟢
  * Neutral 🟡
  * Negative 🔴
* Helps understand market mood and impact

---

### 💬 AI Chatbot (Core Feature)

* Users can ask anything about stocks:

  * “Should I buy Reliance?”
  * “What is the trend of TCS?”
* Chatbot provides:

  * Prediction
  * Buy/Sell suggestion
  * Risk level
  * Clear explanation (why)

---

## ⚙️ How It Works

1. System collects stock data (historical/demo)
2. AI analyzes data using indicators and logic
3. Predictions and insights are generated
4. Chatbot uses this data to answer user queries
5. Results are displayed through a clean UI

---

## 🎯 Purpose of the Project

* Simplify stock market understanding
* Demonstrate AI usage in finance
* Provide a beginner-friendly analysis tool
* Combine multiple features into one platform

---

## ⚠️ Current Limitations

* Uses **demo / offline data** (no live API integration yet)
* Predictions are **basic and for learning purposes only**
* Not intended for real financial advice

---

## 🔮 Future Scope

* Integration with live market APIs
* Advanced ML models (LSTM, Deep Learning)
* Real-time alerts and notifications

## 📚 Conclusion

NeuraWealth demonstrates how AI can be used to **analyze, predict, and explain stock market behavior** in a simple and user-friendly way. It brings together multiple components—data, prediction, sentiment, and chatbot—into a single intelligent system.
