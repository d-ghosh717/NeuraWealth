About Neura2

What is Neura2?

Neura2 is an AI-powered Indian stock market advisor. It uses historical price data from Yahoo Finance CSVs to compute technical indicators (RSI, SMA, EMA, MACD) and generate Buy / Hold / Sell signals.


Adding Stock Data

Download CSV files from finance.yahoo.com and place them in data/stocks/

Supported filenames for stocks:

  • WIPRO.NS.csv or wipro.NS.csv
  • TCS.NS.csv, RELIANCE.NS.csv, etc.

Adding Index Data (Nifty / Sensex)

Place index CSVs in the same data/stocks/ folder:

Index Filename Yahoo Symbol
Nifty 50 ^NSEI.csv ^NSEI
Sensex ^BSESN.csv ^BSESN
Nifty Bank ^NSEBANK.csv ^NSEBANK
